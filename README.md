# TidalSwift

Tidal Music Streaming Client & Library written in Swift

It supports all major features of the official Tidal app, while adding additional ones, like New Releases, Lyrics, Dark Mode, Downloads & Offline Playback â€“ all while being only 1/10th the size of the official app.
